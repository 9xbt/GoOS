# Security Policy

## Reporting a Vulnerability

Goto https://discord.owen2k6.com/ and open a staff help ticket, otherwise use the issues tab.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.4.1-beta3   | :white_check_mark: |
| 1.4.1-beta3  | :x: |
| 1.4.1-beta2         | :x:     |
| 1.4.1-beta1   | :x: |
| 1.4           | :white_check_mark:                |
